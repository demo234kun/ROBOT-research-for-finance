# 需求文档

## 1. 应用概述

### 1.1 应用名称
RobotResearch - 机器人产业链投研分析平台

### 1.2 应用描述
面向机器人赛道的轻量级投研工具,聚焦产业链知识图谱、机构深度研报重制、以及东方财富实时事件流三大核心模块,帮助用户快速获取行业信息、理解产业链结构、阅读专业研报。

### 1.3 设计理念
「做减法」——每个模块保持简单、独立,方便快速验证和迭代。采用深色金融风格UI,强化专业感。

## 2. 用户与使用场景

### 2.1 目标用户
- 机器人行业投资分析师
- 二级市场研究员
- 关注机器人赛道的个人投资者
- 产业研究人员

### 2.2 核心使用场景
- 快速了解机器人产业链结构及上市公司分布
- 实时追踪机器人行业热点事件与新闻动态
- 高效阅读机构研报,理解专业术语与核心观点
- 查询机器人领域专业名词释义

## 3. 页面结构与功能说明

### 3.1 页面层级结构

```
首页/导航页
├── 产业链图谱页
│   └── 节点详情面板(右侧)
├── 事件流页
├── 研报列表页
│   └── 研报详情页
└── 术语库页
```

### 3.2 页面功能说明

#### 3.2.1 首页/导航页
- 展示品牌Logo
- 展示应用简介
- 提供三大模块入口:
  - 产业链知识图谱
  - 实时事件流
  - 深度研报重制
- 提供术语库入口

#### 3.2.2 产业链图谱页

**页面布局**
- 左侧:全屏树图可视化区域
- 右侧:节点详情面板

**树图可视化**
- 展示三级树状结构:
  - 一级节点:上游核心零部件、中游本体制造、下游系统集成与应用
  - 二级节点:
    - 上游:减速器、伺服系统、控制器、传感器、末端执行器
    - 中游:工业机器人、服务机器人、特种机器人、人形机器人
    - 下游:汽车制造集成、3C电子集成、物流仓储集成
  - 三级节点:各二级节点下挂的上市公司
- 支持节点折叠/展开
- 点击节点触发详情面板展示

**节点详情面板**
- 展示节点名称
- 展示节点描述
- 展示关联股票列表:
  - 股票名称
  - 股票代码(A股/港股/美股)
  - 关联强度标签(核心/一般/概念)

**搜索功能**
- 支持搜索板块名称或股票代码
- 搜索结果高亮显示匹配节点

#### 3.2.3 事件流页

**时间线视图**
- 按时间倒序展示事件列表
- 每条事件包含:
  - 标题
  - 发布时间
  - 来源媒体
  - 原文链接(点击跳转)

**筛选功能**
- 提供关键词筛选输入框
- 支持筛选关键词示例:「减速器」「人形机器人」「伺服系统」等
- 筛选后实时更新事件列表

**分页功能**
- 支持分页浏览历史事件
- 每页展示固定数量事件

#### 3.2.4 研报列表页

**研报卡片列表**
- 每张卡片展示:
  - 研报标题
  - 发布机构
  - 发布日期
  - 标签(如「人形机器人」「减速器」)

**筛选器**
- 按标签筛选
- 按机构筛选

**交互**
- 点击卡片进入研报详情页

#### 3.2.5 研报详情页

**页面顶部**
- 展示研报标题
- 展示一句话导读
- 展示发布机构、发布日期

**内容区域**
- 关键论点以卡片形式呈现,数字高亮显示
- 表格数据转换为可排序表格或柱状图展示
- 时间线组件展示行业里程碑事件
- 专业术语自动匹配:
  - 术语以下划线标识
  - 点击术语弹出解释框
- 关联产业链节点与股票代码(可点击跳转至图谱页对应节点)

#### 3.2.6 术语库页

**搜索功能**
- 提供搜索输入框
- 支持模糊搜索术语名称

**术语列表**
- 展示术语名称
- 展示术语解释
- 按字母或分类排序

## 4. 业务规则与逻辑

### 4.1 数据来源规则

**产业链图谱数据**
- 使用内置JSON静态数据
- 数据结构包含节点ID、名称、描述、父节点ID、关联股票列表

**事件流数据**
- 通过后端定时任务抓取东方财富「机器人」相关新闻
- 数据存储于数据库
- 前端调用API获取分页数据

**研报数据**
- 使用内置Demo数据(JSON格式)
- 包含2-3篇完整研报示例
- 数据结构包含标题、机构、日期、标签、导读、正文内容(含论点、表格、时间线、术语标记)

**术语库数据**
- 内置约50+条机器人专业名词
- 数据结构包含术语名称、解释、相关领域

### 4.2 API调用规则

**事件流API**
- 端点:GET /api/v1/events
- 参数:page(页码)、keyword(关键词筛选)
- 返回:事件列表、总数、当前页码

**产业链图谱API**
- 获取完整树结构:GET /api/v1/graph/tree
- 获取节点详情:GET /api/v1/graph/node/{id}
- 搜索:GET /api/v1/graph/search?q={关键词}

**研报API**
- 获取研报列表:GET /api/v1/reports?tag={标签}&institution={机构}
- 获取研报详情:GET /api/v1/reports/{id}

**术语库API**
- 查询术语:GET /api/v1/glossary?q={关键词}

### 4.3 术语匹配规则

- 研报详情页加载时,自动扫描正文内容
- 匹配术语库中的专业名词
- 匹配成功的术语添加下划线样式
- 点击术语触发弹窗展示解释

### 4.4 图谱节点关联规则

- 研报详情页中提及的股票代码或板块名称可点击
- 点击后跳转至产业链图谱页
- 自动定位并高亮对应节点

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| API请求失败 | 展示错误提示,提供重试按钮 |
| 事件流无数据 | 展示「暂无相关事件」提示 |
| 搜索无结果 | 展示「未找到匹配结果」提示 |
| 研报列表为空 | 展示「暂无研报数据」提示 |
| 术语库搜索无结果 | 展示「未找到该术语」提示 |
| 图谱节点无关联股票 | 详情面板展示「暂无关联股票」 |
| 研报详情页无术语匹配 | 正常展示正文,不影响阅读 |
| 分页超出范围 | 自动跳转至最后一页或第一页 |

## 6. 验收标准

1. 用户进入首页,点击「产业链知识图谱」入口,进入图谱页,查看完整三级树状结构
2. 用户点击任意二级节点(如「减速器」),右侧详情面板展示节点描述及关联股票列表(含股票代码、关联强度)
3. 用户返回首页,点击「实时事件流」入口,查看按时间倒序排列的机器人行业新闻列表
4. 用户在事件流页输入关键词「人形机器人」进行筛选,列表实时更新为相关事件
5. 用户返回首页,点击「深度研报重制」入口,进入研报列表页,选择任意研报卡片进入详情页
6. 用户在研报详情页查看一句话导读、关键论点卡片、可视化图表,点击下划线术语查看解释弹窗
7. 用户点击研报中的股票代码,跳转至产业链图谱页并自动定位高亮对应节点
8. 用户进入术语库页,搜索「减速器」,查看术语解释
